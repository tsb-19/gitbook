Git

Git

多仓库

  • 将另外一个远程仓库设置为 remote：git remote add origin url
  • 查看本地已经关联的仓库： git remote -v
  • 删除远程库 git remote rm origin
  • 在本地新建一个与远程仓库对应的分支：git checkout -b dev upstream/dev
  • 将更新推送到自己的远程仓库：git push origin ch4
